# 学习自律计划

内测版本：v0.1.10

这是一个纯前端学习规划工具，面向小学家庭学习场景，用来安排每周学习计划、打印打勾、导出留档。

## 在线/本地运行

这是一个静态网页应用，不需要后端服务。

```bash
python3 -m http.server 4177
```

然后打开：

```text
http://127.0.0.1:4177/
```

也可以直接双击 `index.html` 本地使用。

## 内测定位

- 不需要安装软件，解压后直接打开 `index.html`
- 数据保存在当前浏览器本地，不上传到服务器
- 适合先发给少量家长体验，收集真实使用反馈
- 当前重点验证：学习内容库、周计划打印、Excel/PDF 导出是否顺手

## 当前能力

- 录入孩子信息、本周主题和本学期周次
- 默认生成一周计划，包含可删除的学校作业
- 每天默认包含“学校作业”，任意日期都可以删除，假期也可以一键清空
- 运动和语文、数学、英语一样通过左侧选择添加
- 支持设置本周周一日期，并在每天显示月日
- 支持记录本学期第几周
- 支持复选周一至周日，把同一条学习内容批量添加到多个日期
- 支持按学科批量添加自定义学习内容，并在统一总览中分学科批量删除
- 默认学习内容只保留“其他”，常用内容通过内容库自行添加
- 支持其他内容手写、课外课程和上课时间标注
- 完成情况使用可打印的小方框选项
- 支持手动删除和修改任务
- 自动统计本周学习量和学科分布
- 支持浏览器打印，打印时会隐藏编辑控件并尽量压缩到一页
- 周计划页面用日期色和学科色做区隔，打印时更容易看清
- 支持导出适合打印的 Excel 周表：周一至周日为列，学习内容按学科向下排列，并用日期色和学科色区隔
- 课外课程会完整显示课程时间，PDF 打印样式默认横版且尽量不留浏览器页眉页脚空间
- Excel 和 PDF 都恢复为接近页面的周卡片样式，保留日期色、学科色和课外课程完整时间
- 本周周一日期可自动应用，也可以点击“确定”立即刷新周一至周日日期
- 假期可一键清空默认学校作业；学校作业也可按细项手动添加，例如语文默写、数学订正、英语听读
- 支持通过浏览器打印另存为 PDF，版式尽量保持页面样式

## 使用方式

1. 解压 zip 文件。
2. 双击打开 `index.html`。
3. 在左侧填写孩子信息、本周信息和学习内容库。
4. 添加每天学习内容后，可以打印、导出 Excel，或通过浏览器打印另存为 PDF。

## 内测反馈建议

可以重点收集这些问题：

- 家长是否能独立完成一周计划
- 打印版是否够清楚，孩子是否愿意打勾
- 学习内容库是否好用
- 课外课程和运动安排是否够灵活
- 是否需要多孩子、多周历史、内容库导入导出

## 后续扩展建议

1. 资料包后台：维护年级、教材章节、能力点和练习题库。
2. AI 题目识别：拍照识别题目，自动提取知识点、题型和错因。
3. 错题本：把错题归档到知识点，形成“错题-训练题-复测”的闭环。
4. 账号系统：保存多个孩子、多周计划和完成记录。
